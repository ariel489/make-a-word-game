🎮 Make a Word Game

Online multiplayer — play with friends anywhere!

How to play:
• Host creates a room and shares the 6-letter code.
• Friends join from anywhere using the code.
• Once everyone's in, host clicks Start. All players see the same 2 letters at the same time.
• You have 12 seconds to type a real English word containing BOTH letters.
• Copy & paste are disabled. Used words can't be repeated.
• First to 2 points (2 players) or 3 points (3 players) wins!
